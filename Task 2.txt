Document Object :1.  The document object represent a web page that is loaded in the browser.2. By accessing the document object, we can access the element in the HTML page.3. With the help of document objects, we can add dynamic content to our web page.4. The document object can be accessed with a window.document or just document.5. It is loaded inside the window. It is the object of window property.6. All the tags, elements with attributes in HTML are part of the document.7. Syntax  :  document.property_name;8. Example : document.title :  will return the title of the documentWindow Object :1. The window object is the topmost object of the DOM hierarchy.2. It represents a browser window or frame that displays the contents of the webpage.3. Whenever a window appears on the screen to display the contents of the document, the window object is created. 4. It is the very first object that is loaded in the browser.5. It is the object of the browser.6. Global objects, functions, and variables of JavaScript are members of the window object.7. Syntax: window.property_name;8. Example : window.innerHeight : will return the height of the content area of the browser
